# Contributing

Contributions are welcome.

Good first improvements:

- Better controller mappings
- Sensitivity/deadzone presets
- Lower-latency packet format
- Cleaner installer
- Better error messages
- More themes

Before submitting:

1. Make sure the project builds.
2. Do not commit `bin/`, `obj/`, `.vs/`, or publish output.
3. Do not include personal logs, device IDs, or screenshots with private info.
