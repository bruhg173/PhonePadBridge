# Privacy

PhonePad Bridge is intended to be local-first.

## What the app does

- Starts a local web server on your PC.
- Receives controller packets from your phone.
- Updates a virtual Xbox controller.
- Shows connection status in the local desktop GUI.

## What the app does not do

- It does not send analytics.
- It does not upload logs.
- It does not require an account.
- It does not use a cloud API.
- It does not contain API keys.
- It does not intentionally store personal data.

## Things to avoid posting publicly

If you share screenshots or logs, they may contain:

- Local IP addresses
- Android device serial/status from ADB
- Your selected port
- Controller/browser identification strings

These are usually low-risk, but you may still prefer to blur them.
